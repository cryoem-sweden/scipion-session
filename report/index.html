5) Data Access

We kindly ask you to download your data as soon as possible and confirm with your contact person at the Cryo-EM facility that your data can be deleted. The download server is data.cryoem.se and currently we recommend to use rsync + ssh for downloading your data. For latest instructions please log in to the server with ssh, using the username and password that has been provided to you. 

$ ssh USERNAME@data.cryoem.se (ssh cem00090@data.cryoem.se)

A client that supports the SFTP protocol should work too. Settings for e.g. FileZilla are: 

Host: sftp://data.cryoem.se
Port: 22
Username & password as provided to you

Just to remind you what is stored in which folder:

/home/cem00090/data/
	cem00090_atlas_20170317 contains the EPU output for the atlas acquisition
	cem00090_grid3_epu_20170317 contains the EPU output for the image acquisition
	k2_frames contains raw data as single frames

6) Data pre-processing

You will find the preprocessed data in folder:

/home/cem00090/data/cem00090_scipion/Runs

	000002_ProtImportMovies contains stacked movie frames

	000088_ProtMotionCorr 
		contains motion corrected averages of the movie frames 
		(frames_aligned_mic.mrc for non dose-weighted average, which we use for CTF estimation 
		and frames_aligned_mic_DW.mrc for dose-weighted average, which we use for 2D and 3D analysis)

	000185_ProtCTFFind contains CTF estimates from CTFFind4 
      aborted
	000280_ProtCTFFind contains CTF estimates from CTFFind4 
      option 1(settings: resolution 0.05 to 0.11)
	000375_ProtCTFFind contains CTF estimates from CTFFind4 
      option 2 (settings: resolution 0.05 to 0.17)
	000235_ProtMonitorSummary contains the summary of pre-processing for 000185_ProtCTFFind
	000330_ProtMonitorSummary contains the summary of pre-processing for 000375_ProtCTFFind
	000425_ProtRelionExportCtf contains export file for ctf parameters from option1/000280_ProtCTFFind
	000452_ProtRelionExportCtf contains export file for ctf parameters from option2/000375_ProtCTFFind

Note 1: If you want to run Relion for data processing, you may want to run it above those directories from the /Run folder.

Note 2: You may want to redo the CTF estimation using your own settings. You could also use the micrographs_ctf_000452.star or micrographs_ctf_000425.star file, which you could directly import into relion. Please note, that you may want to exchange the non-DW file with the DW-file micrographs.
